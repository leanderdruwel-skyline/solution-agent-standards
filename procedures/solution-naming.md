# Solution Name

## Scope

These guidelines apply to all agents working on **solution repositories**. If the repository name starts with `ldr-`, agents **must** read and follow this file in addition to the [global instructions](global-instructions.md).

> **Note**: This file uses severity tags `[ERROR]`, `[WARNING]`, and `[INFO]` as defined in [global-instructions.md](global-instructions.md#severity-levels).

---

## Naming Conventions

- **[ERROR]** The name of every file in the Files folder must start with `SOL-` to indicate that it is a solution repository. If that is not the case, report an error that blocks the pipeline.
