# Solution Name

## Scope

These guidelines apply to all agents working on **solution repositories**. If the repository name starts with `ldr-`, agents **must** read and follow this file in addition to the [global instructions](global-instructions.md).

---

## Naming Conventions

- The name of every file into the Files folder must start with `SOL-` to indicate that it is a solution repository. If that is not the case, report an error that the file names are incorrect.
